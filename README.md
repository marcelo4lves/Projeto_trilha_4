# Projeto de E-Comerce para a disciplina Desenvolvimento de Software I
Uma sistema de e-commerce utilizando o React

# Descrição do Projeto
O projeto consome dados  de uma API externa e permite que o usuário
- Veja todos os produtos disponíveis
- Pesquise um produto pelo nome
- Pesquise produtos por categoria
- Veja a página do produto individual
- Adicione produtos no carrinho
-Altere a quantidade dos itens e visualize o valor total
-Faça o check-out do pedido

# Tecnologias utilizadas
- React 
- React Router DOM 
- Vite 
- JavaScript (ES6+)
 - HTML5
 - CSS3 
- Context API 
- Fake Store API 
- Fetch API

# Como executar o projeto

### 1. Clone o repositório
```bash 
git clone https://github.com/marcelo4lves/Projeto_trilha_4.git
 ``` 

### 2. Instale as dependências 
```bash npm install ``` 
### 3. Execute o projeto 
```bash 
npm run dev
```
